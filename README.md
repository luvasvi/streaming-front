💻 FTTB - Streaming Interface (Front-end)
Interface responsiva e de alta fidelidade desenvolvida em Angular 19, focada em uma experiência de streaming moderna, fluida e personalizada.

🛠 Tecnologias e Arquitetura
Angular 19: Uso de Standalone Components para uma estrutura leve e modular.

RxJS: Gerenciamento reativo de chamadas assíncronas e fluxos de dados.

Angular Control Flow: Implementação de @if e @for para renderização condicional otimizada.

CSS3 Personalizado: Design avançado com Glassmorphism, iluminação Neon e animações de scroll suaves.

🚀 Funcionalidades e Segurança
Sistema de Autenticação: Proteção de rotas com canActivate e persistência de sessão via localStorage.

Gestão de Favoritos: Integração bidirecional com o back-end para salvar e sincronizar a "Minha Lista" por usuário.

Feedback do Usuário: Sistema de Toast Notifications interno (sem alertas de navegador) para confirmações e erros.

Busca Global: Filtros dinâmicos por títulos, gêneros e elenco através da API do TMDB.

📺 Destaques Visuais
Layout de Temporadas: Seleção interativa de temporadas com carregamento dinâmico de metadados.

Lista de Episódios: Design otimizado com truncamento de texto (line-clamp) e tratamento de miniaturas.

Skeleton Loading: Interface fantasmagórica que melhora a percepção de performance durante o carregamento de dados.

🔧 Instalação e Execução
Certifique-se de ter o Node.js e o Angular CLI instalados.

Clone o repositório e execute:

Bash
npm install
ng serve
Acesse a aplicação em http://localhost:4200.
