# 💻 FTTB - Streaming Interface (Front-end)

Interface responsiva desenvolvida em **Angular** para uma experiência de streaming moderna.

### 🛠 Tecnologias
- **Angular 19** (Standalone Components).
- **RxJS**: Gerenciamento de chamadas assíncronas.
- **CSS3 Personalizado**: Efeito Glassmorphism, Neon e animações de scroll.

### 📺 Destaques Visuais
- **Layout de Temporadas**: Seleção interativa de temporadas com carregamento dinâmico.
- **Lista de Episódios**: Design com truncamento de texto (`line-clamp`) e miniaturas.
- **Skeleton Loading**: Melhora a experiência do usuário durante o carregamento de dados.
